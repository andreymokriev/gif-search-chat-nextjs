
# Чат с поиском GIF анимаций

Чат с возможностью отправки анимаций GIF, написанный на nexjs и использующий Giphy API


## Использование
- Интерфейс активируется, когда вводимый текст начинается с комбинации /gif
- Текст после ключевой комбинации и пробела используется в качестве запроса для поиска подходящих gif изображений
- После выбора изображение отображается над полем ввода как отправленное
